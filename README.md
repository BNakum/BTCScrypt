BTCScrypt
=========

Objective-C interface to Colin Percival's Scrypt with progress block.

#### Usage:

1. Drop all the files in your project.
2. Check only one of scrypt-*.c files (e.g. scrypt-nosse.c).
3. Run BTCScrypt().
